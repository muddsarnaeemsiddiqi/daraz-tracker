# Daraz Price Drop Alert

A Chrome extension to track product prices on Daraz.pk and get notified when prices drop.

## Features
- Track product prices from Daraz.pk
- Get browser notifications when prices drop
- Support for multiple variants/colors
- Visual price history with charts
- Premium system with email-based access

## How to Install
1. Go to `chrome://extensions/`
2. Enable **Developer mode** (top right)
3. Click **Load unpacked**
4. Select the `public` folder

## How to Use
1. Go to any Daraz.pk product page
2. Click the extension icon in Chrome toolbar
3. Click "Track Current Page"
4. Monitor price changes in the extension
